# Culturaft

Full-stack e-commerce sample (Culturaft).

Backend:

```powershell
cd backend
npm install
copy .env.example .env
# edit .env if needed
npm run seed   # creates admin user and sample products
npm run dev
```

Frontend:

```powershell
cd frontend
npm install
npm run dev
```
